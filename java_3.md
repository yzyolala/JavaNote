## Java 字符串和数组

### 1. String 和数组的不可变性

- String是不可变的（与数组一样）

### 2. 引号使用

- String是双引号，字符是单引号

### 3. String 的创建方式

- `new String("hello");`  //使用new关键字在heap上创建
- `String a = "hello";`  //保存在字符串常量池中

### 4. 数组的大小

- 数组的大小一旦创建就不能改变

### 5. 数组的内存储存

- 数组将数据存储在连续的内存中

### 6. 数组的创建方式

- `int[] arr = new int[2];`
- `int[] marks = {10, 20, 30, 16};`

### 7. 数组的变量类型

- 数组存储的变量都是一样的，同质的

### 8. 列表和数组

- Java中的列表就是ArrayList，大小可变，数组大小一旦创建就不可变

### 9. 构造方法

- 与class名字相同（
在Java中，构造方法的名称必须与所在类的名称完全相同。这是Java的语法规定，无法更改。如果构造方法与所在类的名称不同，则编译器将无法识别它，并会将其视为普通方法，而不是构造方法。）
- 有参数列表或者无参数列表
- 没有返回值
